🎤 Interview Answer — Use this framework

"Fiori Catalog creation has three main phases:"
Phase 1 – Design: Open Launchpad Designer (/UI2/FLPD_CUST), create a catalog, search for the SAP business catalog and drag it in as a reference — this creates the tile.
Phase 2 – Layout: Use /UI2/FLP to create a Space and Page so the tile appears on the Launchpad.
Phase 3 – Authorization: In PFCG, create a role, add the Space, then add IWSG (frontend) and IWSV (backend) OData services to generate the S_SERVICE authorization objects. Save, generate, and assign to user.
Every step must be linked to a Transport Request before making changes.
